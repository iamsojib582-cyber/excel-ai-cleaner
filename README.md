# 🚀 Excel AI Cleaner

**AI-powered Data Analyst Studio** for automated Excel/CSV cleaning and preprocessing.

Built with Python and Groq (Llama 3), this desktop application helps analysts transform messy datasets into clean, structured data in seconds.

---

## 🔥 Key Features

* 🤖 **AI Data Audit (Llama 3)**
  Detects missing values, wrong data types, duplicates, inconsistencies, and anomalies

* ⚡ **One-Click Cleaning**
  Automatically fixes:

  * Null values
  * Data types (text → numeric/date)
  * Duplicates
  * Formatting issues

* 📂 **Smart File Loader**
  Supports:

  * CSV (auto encoding detection: UTF-8, Latin-1, etc.)
  * Excel (.xlsx, .xls)
  * JSON

* 📊 **Built-in Analytics Tools**

  * Pivot table generator
  * Data profiling
  * Summary statistics

* 📈 **Visualization**

  * Bar, Line, Pie, Histogram, Scatter charts

* 🧹 **Manual Cleaning Tools**

  * Replace values
  * Trim spaces
  * Fix dates & numbers
  * Remove junk rows

* 💾 **Export Options**

  * Cleaned CSV
  * Excel files
